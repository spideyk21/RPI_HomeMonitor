Raspberry Pi Home Monitor
==========
Raspberry Pi based Home Monitor. Simiar to D-Link Wifi Smart Plug.
Includes:
-Wifi Connected
-Motion Sensing
-Temerature Sensing
-Supports 120VAC Outlet (Future)

###Hardware:
-Raspberry Pi
-SD Card
-Enclosure (3d Printed)
-Wifi Module
-PIR Sensor
-Led's
-Button/Switch
-Battery
-Battery Charger
-Relay (Optional/Future)

###Software:
-Raspbian (latest)
